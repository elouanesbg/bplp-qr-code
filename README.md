# شرح مشروع BPLP QR Code
## دليل التخرج لطلبة قسم علم المكتبات

---

## 📖 مقدمة عن المشروع

مشروع **BPLP QR Code** هو تطبيق ويب مبتكر يهدف إلى **رقمنة وتنظيم المكتبات الرقمية** من خلال استخدام **رموز QR** والتقنيات الحديثة. يُمكن المستخدمين من الوصول إلى معلومات الكتب بسهولة عبر مسح رموز QR أو البحث المباشر.

---

## 🎯 الأهداف الرئيسية

| الهدف                    | الوصف                                       |
| ------------------------ | ------------------------------------------- |
| **رقمنة المحتوى**        | تحويل معلومات الكتب إلى صيغة رقمية منظمة    |
| **سهولة الوصول**         | إمكانية البحث والوصول السريع لمعلومات الكتب |
| **استخدام تقنيات حديثة** | استخدام رموز QR لتسهيل الوصول للمحتوى       |
| **توفير واجهة سهلة**     | تصميم جذاب وسهل الاستخدام                   |
| **التنظيم والفهرسة**     | تنظيم الكتب حسب تصنيفات معينة               |

---

## 🏗️ مكونات المشروع الرئيسية

### 1. **الواجهة الرسومية (User Interface)**
الواجهة هي ما يراه المستخدم عند زيارة التطبيق. تتكون من:

- **الصفحة الرئيسية**: تعرض قائمة بالكتب المتاحة
- **صفحة تفاصيل الكتاب**: تعرض معلومات كاملة عن كتاب واحد
- **صفحة الرفوف**: تعرض تجميعات من الكتب (مثل الأرفف في المكتبة الحقيقية)

### 2. **قاعدة البيانات**
تخزن معلومات الكتب مثل:
- العنوان والمؤلف
- الناشر وسنة النشر
- رقم ISBN (معرّف فريد للكتاب)
- وصف الكتاب والكلمات المفتاحية
- صورة الغلاف

### 3. **نظام التوجيه (Routing)**
يسمح للمستخدم بالانتقال بين الصفحات المختلفة باستخدام روابط مثل:
- `https://bplp-medea.web.app/` - الصفحة الرئيسية
- `https://bplp-medea.web.app/book/{isbn}` - صفحة كتاب معين
- `https://bplp-medea.web.app/shelf/{id}` - صفحة رف معين

### 4. **رموز QR**
كود ثنائي الأبعاد يمكن مسحه بهاتف ذكي، يحتوي على رابط يوجه المستخدم مباشرة إلى صفحة الكتاب.

## 📚 هيكل بيانات الكتاب

كل كتاب في قاعدة البيانات يحتوي على المعلومات التالية:

```
{
  "isbn": "9789931030027-1",           // معرّف فريد للكتاب
  "title": "نور على نور",              // العنوان الرئيسي
  "title_secondary": "في مجال القرآن", // العنوان الثانوي
  "language": "اللغة العربية",          // لغة الكتاب
  "author": "محمد الصالح الصديق",      // المؤلف
  "publish_country": "الجزائر",        // دولة النشر
  "publisher": "دار هومه",             // الناشر
  "publish_year": 2014,                 // سنة النشر
  "edition": 1,                         // رقم الطبعة
  "keywords": "الإسلام، الربا",        // الكلمات المفتاحية
  "description": "بين أن الله...",    // وصف الكتاب
  "cover": { "url": "..." }             // رابط صورة الغلاف
}
```

## 🎨 واجهات المستخدم

### الصفحة الرئيسية (Home Page)
- عرض قائمة بجميع الكتب المتاحة
- إمكانية البحث عن كتاب معين
- عرض صور أغلفة الكتب

### صفحة تفاصيل الكتاب (Book Page)
عند اختيار كتاب، تظهر صفحة تحتوي على:
- صورة الغلاف كاملة
- المعلومات الكاملة: العنوان، المؤلف، الناشر
- سنة النشر والطبعة
- وصف الكتاب
- الكلمات المفتاحية (Tags)
- رقم ISBN

### صفحة الرفوف (Shelf Page)
- تجميع الكتب حسب تصنيفات (مثل الأرفف الفيزيائية)
- كل رف يحتوي على مجموعة من الكتب ذات الصلة

## 🔄 كيف يعمل التطبيق؟

### خطوات الاستخدام من وجهة المستخدم:

```
1. المستخدم يزور موقع التطبيق
   ↓
2. يرى قائمة بالكتب المتاحة
   ↓
3. يختار كتاباً أو يمسح رمز QR
   ↓
4. يظهر صفحة بمعلومات الكتاب التفصيلية
   ↓
5. يمكنه العودة للقائمة أو استكشاف كتب أخرى
```

### من جهة النظام:

```
المستخدم يطلب رابط الكتاب
   ↓
التطبيق يبحث عن البيانات في قاعدة البيانات
   ↓
يسترجع معلومات الكتاب
   ↓
يعرضها في واجهة جميلة
   ↓
يظهرها للمستخدم
```

## 🎓 الدروس المستفادة (Learning Outcomes)

من خلال هذا المشروع، يمكن لطالب علم المكتبات أن يتعلم:

1. **الرقمنة والتحول الرقمي**: كيفية تحويل المعلومات التقليدية إلى صيغة رقمية
2. **إدارة البيانات**: كيفية تنظيم وتخزين المعلومات بشكل فعال
3. **تقنيات البحث**: استخدام الكلمات المفتاحية والفهرسة
4. **تطبيقات الويب**: فهم أساسيات تطبيقات الويب وواجهات المستخدم
5. **البيانات الوصفية (Metadata)**: أهمية المعلومات المرفقة بالموارد
6. **الوصول المفتوح**: توفير سهولة الوصول للمعلومات
7. **التكنولوجيا الحديثة**: استخدام تقنيات مثل QR والتطبيقات السحابية

## 📋 الخلاصة

مشروع BPLP QR Code هو تطبيق عملي يجمع بين:
- **المعرفة المكتبية**: تنظيم وفهرسة المعلومات
- **التقنية الحديثة**: استخدام الويب والتطبيقات السحابية
- **سهولة الوصول**: توفير معلومات منظمة يسهل الوصول إليها

يمثل هذا المشروع نموذجاً لكيفية تطبيق تقنيات المعلومات في مجال علم المكتبات، وكيفية أن تلعب المكتبات دوراً مهماً في العصر الرقمي.
